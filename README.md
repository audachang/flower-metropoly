# 🌸 花花大富翁 (Flower Metropoly)

用花作主題的大富翁遊戲。把荒地一瓣一瓣種成花園：買種子、種花瓣、綻放完整花，挑戰稀有變異。

## ▶ 線上遊玩
https://audachang.github.io/flower-metropoly/

## 玩法
- 每格是一種花的種子，像買房子一樣可以買下種子（取得地權）。
- 蓋房子 = 種花瓣：1 瓣花苞 → 2 瓣半開 → **3 瓣綻放完整花** → **第 4 瓣稀有變異**（擲育種骰決定成敗）。
- 壟斷一整色花圃才能開始種花瓣。
- 每回合擲骰後只能做一個動作（買種子或種一片花瓣），做完自動換下一位。

## 特色
- 40 格棋盤，開場可選 1–3 位人類玩家 + 0–3 台電腦對手（穩健型 AI）。
- 陽光卡 / 蟲害卡抽卡系統、蜜蜂市集、傳送門、季節轉換。
- 升級版枯萎花園：隨機際遇、被困時收租領補償、擲雙數或付贖金脫困。
- 兩座中央祕密花園（彩虹花園 🌈 / 黃金花園 ✨），抽到祕密地圖卡才能進入。
- 棋子逐格移動動畫、滑鼠滑過格子顯示完整資訊、爵士背景音樂（本機 MP3 / YouTube / 內建合成）。
